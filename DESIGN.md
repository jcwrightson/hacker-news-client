# Hacker News Client: Design Doc
## Author: J. C. Wrightson


# Brief:

Using a TDD process, create a vanilla javascript client to fetch Hacker News stories. When a user selects a story, load it's comments.

# Required functions:

- Fetch Top Stories
- Fetch Story
- Fetch Comments
- Render Top Stories
- Render Comments

# Data Structure

- Store
	- Stories: []
